RFQ Translation Workbench - Windows online bootstrap

1. Read README.md, docs/INSTALL_WINDOWS.md, PRIVACY.md and THIRD_PARTY_NOTICES.md.
2. Verify this ZIP with the published SHA256SUMS.txt.
3. Extract the ZIP to a local folder.
4. Run Install.cmd as your normal Windows user.
5. Configure your own model endpoint and API key when prompted by the documentation.

The installer downloads Python and fixed dependencies from official upstream sources.
It does not enable network sharing, autostart, firewall rules or router changes.
