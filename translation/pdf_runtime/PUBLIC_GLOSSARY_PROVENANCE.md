# 公共默认术语词表来源与权属

## 定位

`rfq_default_glossary.json` 是 RFQ Translation Parameter Tool 从零整理的公开通用术语映射，用于稳定泵、机械、材料、压力、流量和工程标准类短文本的中文译法。

该词表不包含、也不得加入：

- 客户、业主、顾问、供应商或项目名称；
- 内部简称、真实位号、文件编号或路径；
- 从业务文件复制的标题、段落或整句译文；
- 受保密、著作权或合同限制的专有词表内容。

## 来源与权属

当前词条由本项目根据通用机械工程概念独立整理，并以短术语映射形式维护。标准号仅作为不可翻译的通用标识保护，不转载标准正文，也不表示获得任何标准组织授权。

项目维护者可在符合仓库许可证和第三方合规要求的前提下修改或再分发本词表。具体软件许可证与第三方组件声明以仓库许可证文件和 `deploy/THIRD_PARTY_NOTICES.txt` 为准。

## 维护规则

1. 只收录跨项目可复用的通用短术语。
2. 新词条必须能说明通用工程含义，不以某份历史文件作为唯一依据。
3. 标准号、型号、单位和代码默认保持原样；仅翻译其通用说明文字。
4. 发现多种行业译法时，优先采用泵及机械行业常用、紧凑且适合表格单元格的表达。
5. 不确定权属或可能识别具体业务资料的词条不得进入公共词表。

## 私有扩展边界

部署方可通过 `B_PDF_TRANSLATION_PRIVATE_GLOSSARIES` 环境变量加载供 PDF、Word 和 Excel 共用的本机 CSV/JSON 私有词表，也可用 wrapper 的 `--glossary-files` 参数仅为一次 PDF 调试加载。私有词表由部署方自行维护和授权，不属于公共默认词表，也不得提交到 Git。

运行证据只记录私有词表是否启用、文件数、词条数和不可逆摘要签名，不记录原路径或词条内容。匿名运行副本会在翻译结束后清理。
